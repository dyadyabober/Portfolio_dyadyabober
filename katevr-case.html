Огляд проекту Project Overview

KateVR — це комплексний лендінг для VR-технологій, який я створив повністю з нуля за дизайном з Figma в рамках навчання в Mate Academy. KateVR is a comprehensive VR technology landing page that I created completely from scratch based on a Figma design as part of my studies at Mate Academy. The project is classified as HARD difficulty level and demonstrates deep frontend development skills.

🌐 Мультимовність Multilingual Support

Реалізація динамічного перемикання між українською та англійською мовами з збереженням стану інтерфейсу та плавними переходами. Implementation of dynamic switching between Ukrainian and English languages while preserving interface state with smooth transitions.

function setLanguage(currentLanguage) {
  const notCurrentTagNames = document.querySelectorAll('[data-lang]');
  
  notCurrentTagNames.forEach(function(tag) {
    if (!tag.classList.contains('hidden-lang')) {
      tag.classList.add('hidden-lang');
    }
  });

  const currentTagNames = document.querySelectorAll(`[data-lang="${currentLanguage}"]`);
  
  currentTagNames.forEach(function(tag) {
    if (tag.classList.contains('hidden-lang')) {
      tag.classList.remove('hidden-lang');
    }
  });
}
