Огляд проекту

KateVR — це комплексний лендінг для VR-технологій, який я створив повністю з нуля за дизайном з Figma в рамках навчання в Mate Academy. Проект класифікується як HARD рівень складності і демонструє глибокі навички frontend розробки.

Сайт представляє інноваційну VR-систему пересування KAT Loco з повною інформацією про продукт, технічні характеристики, багатокроковий процес покупки та мультимовну підтримку.

Технічні виклики

Проект вимагав вирішення складних технічних задач та імплементації просунутої функціональності.

🌐 Мультимовність

Реалізація динамічного перемикання між українською та англійською мовами з збереженням стану інтерфейсу та плавними переходами.

🎭 Складні анімації

Створення кастомних слайдерів, модальних вікон, hover-ефектів та інтерактивних елементів з плавними CSS та JS анімаціями.

📱 Pixel-perfect адаптив

Ідеальна адаптивність для всіх пристроїв з 4 брейкпоінтами: мобайл (320px+), планшет (768px+), десктоп (1280px+), повний HD (1920px+).

🛒 E-commerce функції

Багатокроковий процес покупки з валідацією форм, вибором країн/міст, розрахунком вартості та обробкою платіжної інформації.

🎯 SPA навігація

Реалізація Single Page Application з hash-роутингом, плавною прокруткою та динамічним управлінням станом без фреймворків.

♿ Доступність

Імплементація ARIA атрибутів, keyboard navigation, семантичної розмітки та focus management для користувачів з обмеженими можливостями.

💡 Мультимовність - динамічне перемикання контенту 
function setLanguage(currentLanguage) {
  const notCurrentTagNames = document.querySelectorAll('[data-lang]');
  
  notCurrentTagNames.forEach(function(tag) {
    if (!tag.classList.contains('unselect-language')) {
      tag.classList.add('unselect-language');
    }
  });

  const currentTagNames = document.querySelectorAll(`[data-lang="${currentLanguage}"]`);
  
  currentTagNames.forEach(function(tag) {
    if (tag.classList.contains('unselect-language')) {
      tag.classList.remove('unselected-language');
    }
  });
}
